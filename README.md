# Legacy version of my personal Website from 2016

- Developed with:
  - Gulp
  - Jade (now Pug)
  - Sass

## View locally

- install php
- start php server on localhost within source directory
- access entrance page on your local machine by directing your browser to `localhost:<port>/entrance.php`
